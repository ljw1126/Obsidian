## 플러그인 
- `google-java-format` ✨
	- 설치 후에 추가 설정 필요
	- [공식 문서](https://github.com/google/google-java-format/blob/master/README.md#intellij-jre-config)에서 설정 복사 후 `[Help > VM option..]` 에 하단 붙여 넣기
	- 재시작
	  
```text
--add-exports=jdk.compiler/com.sun.tools.javac.api=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.code=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.file=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.parser=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.tree=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.util=ALL-UNNAMED
```

## 단축키(shortcut)

**프로젝트 창 포커스 이동** ✨
- `mac` : command + 1
- `window` : alt + 1


> 프로젝트 창 포커스 이동 후 보고 싶은 클래스 파일 미리보기는 `space`를 누르면 된다



**에디터 창 키우기**✨
- `mac` : shift + command + f12
- `window` : ctrl + shift + f12

> 에디터 창 빼고 다 숨기기네


**에디터 창 이동**✨
- `mac` : control + tab
- `window` : ctrl + tab


**새 파일 생성**
- 에디터에서 (동일 위치에 생성)
	- `mac`: control + option + n
	- `window`: ctrl + alt + insert
- 프로젝트 창에서 (포커스 이동 후)
	- `mac` : command + n
	- `window` : alt + insert


**커서 이동**
- 단어별 이동
	- `mac` :  option + 좌우 
	- `window` :  crtl + 좌우
- 라인 시작/끝 이동
	- `mac` : fn + 좌우
	- `window` : home, end
- 페이지 위/아래
	- `mac` : fn + 위아래
	- `window` :  page up, page down


**선택 영역 확장/축소**
- `mac` : option +위아래
- `window` : ctrl + W, ctrl + shift + W  


**주석 처리**✨
- 한 줄 주석
	- `mac` : command + /
	- `window` :  ctrl + /
- 블록 주석
	- `mac` : option + command + /
	- `window` :  shift + ctrl + /

**인덴트**✨
- 추가 : tab 
- 취소 : shift + tab


**자동 인덴트** 
전체 선택 후
- `mac` : control + option + I
- `window` : ctrl + alt + I


**사용처 찾기**
- 찾기 
	- `mac` : option + f7
	- `window` : alt + f7
- 빠른 찾기
	- `mac` : command + B
	- `window` :  ctrl + B
