
## 플러그인 
- `google-java-format` ✨
	- 설치 후에 추가 설정 필요
	- [공식 문서](https://github.com/google/google-java-format/blob/master/README.md#intellij-jre-config)에서 설정 복사 후 `[Help > VM option..]` 에 하단 붙여 넣기
	- 재시작
	  
```text
--add-exports=jdk.compiler/com.sun.tools.javac.api=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.code=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.file=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.parser=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.tree=ALL-UNNAMED
--add-exports=jdk.compiler/com.sun.tools.javac.util=ALL-UNNAMED
```

- `Rainbow Brackets Lite - Free and OpenSource` ✨
	- 코드 편집창에서 중첩된 괄호를 더 구분하기 쉽게 다양한 색으로 표시 
	- 이외에 `Indent Rainbow`도 있음

- `PowerMode2` ✨
	- 편지창의 커서에 특수 효과 표시
	- 커서가 위치한 라인을 돋보이게 한다
	- 설치 후 설정 수정 - 인텔리제이 `Settings`에 `Power Mode` 검색
		- `ENTER POWER MODE`
		- `SPARKS`
		- `VISULIZE EVERY CREATE MOVEMENT`

- `Grep Console`
	- 인텔리제이에서 로그를 볼 떄 더 편리하게 필터링
	- 콘솔 로그에서 원하는 검색어를 `Grep` 하면 다른 탭에서 모아서 보여줌

- `Nginx Configuration`
	- Nginx 설정 편집을 지원

- `.env files support`
	- `.env` 파일 편집을 지원

- `JPA Buddy`
	- JPA 쿼리 메소드 생성 지원 등

- `Docker`
	- 도커 컨테이너와 도커 컴포즈를 지원

- `Spock Framework Support`
	- Spock 테스트 프레임워크를 지원

- `Groovy`
	- Groovy 문법을 지원
